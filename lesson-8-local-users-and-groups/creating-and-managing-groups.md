# Creating and Managing Groups

To create, delete, and modify, these are the following commands:&#x20;

```bash
groupadd: add groups
groupdel: delete groups
groupmod: modify group 

lid -g [GROUP NAME]: list all users that are members of a specific group
id: list the users UID or GID, is also useful for group listing
```

{% hint style="info" %}
usermod is a good resource if you want to add user to a group...read carefully ;)
{% endhint %}
