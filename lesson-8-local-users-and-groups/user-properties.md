# User Properties

User proper are managed in /etc/passwd

* **Name:** the name of the account&#x20;
* **Password:** The secret that is used for authentication may be disabled&#x20;
* **UID:** a unique identifier for users&#x20;
* **GID:** the ID of the primary group
* **GECOS:** Additional non-mandatory information about the user&#x20;
* **Home directory:** the environment where users create personal files (/home)
* **Shell:** The program that will be started after successful authentication
