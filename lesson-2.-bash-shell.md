# Lesson 2. Bash Shell

### What is a shell?&#x20;

* The shell is  an environment where the user interfaces with the OS

### **What is Bash?**&#x20;

* **Bash** (Bourne Again SHell) is a Unix shell and command-line interpreter. It is widely used as the default shell in Linux-based systems and macOS, providing users with an interface to interact with the operating system through commands. It supports scripting capabilities for automating tasks.

### Using Linux Commands

* Commands have common properties&#x20;
* Most commands can be used with args, which further specify the use of the command. Options can be used to modify the behaviour of the command.&#x20;
* For an overview, use **--help** with the command&#x20;
