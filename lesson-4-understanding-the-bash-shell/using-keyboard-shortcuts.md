# Using Keyboard Shortcuts

* **Ctrl-c** quits the current interactive process
* **Ctrl-d** sends an end-of-file character to the current interactive process
* **Ctrl-a** moves the cursor to the start of the current line&#x20;
* **Ctrl-e** moves the cursor to the end of the current line
* **Ctrl-r** reverse search history
* **Ctrl-l** clears screen&#x20;
* **Ctrl-u** removes current line&#x20;
* **Alt-b** moves the cursor one word backward
* **Alt-f** same but forward
