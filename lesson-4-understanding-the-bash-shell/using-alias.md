# Using Alias

Alias can be used to define custom commands&#x20;

* Some commands defined as an alias are provided as the system default
* Type **alias** for an overview of all current aliases
* Use **alias** **key=value** to define your own alias&#x20;
  * **alias del='rm -rf /**'
* Use **unalias** to remove an alias&#x20;
* Custom aliases are stored in the Bash startup files
