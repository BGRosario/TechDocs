# Access Control list (ACL)

What is ACL?&#x20;

* The Access Control List (ACL) provides an additional, more flexible permission mechanism for file systems. It is designed to assist with UNIX file permissions. ACL allows you to give permissions for any user or group to any disc resource.&#x20;

Use if ACL:&#x20;

* Think of a scenario in which a particular user is not a member of a group created by you, but you still want to give them some read or write access. How can you do it without making user a member of group, here comes in a picture ACL.&#x20;
