# Switching User with su

The **su** command is used to switch the current user account a shell environment

* If **su** is used with the - option, the complete environment of the target user is loaded&#x20;
* If **su** is used with the - option, the complete environment of the target user is loaded
* While using **su,** the password of the target user is entered&#x20;
* If the root user has a password, **su -** can be used
* Using **su -** to open a root shell is considered bad practice; use **sudo -i** instead.&#x20;
* The **su** command can help test other user accounts.
